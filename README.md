This is ELRS 3.2.0 for Ebyte E28-2G4T27SX as TX module - connecting to opentx radio as internal module with full duplex only!

This module have HT32F52342 MCU 48Mhz Cortex M0+.

GPIO's - 
PB2 - TX_EN
PB3 - RESET_LORA
PB4 - RX_EN
PC7 - BUSY
PC6 - DIO1
PA5 - DIO2
PA4 - DIO3
PA3 - CSS
PA15 - SKY_POWER
SPI1
